# 🍽️ 바이브코딩 오마카세

> 치수쌤이 추천하는 오늘의 웹앱을 만나보세요.

교육뮤지컬 꿈꾸는 치수쌤이 직접 만든 바이브코딩 웹앱을 여섯 가지 코스로 소개하는 메뉴판 사이트입니다.

---

## 소개

오마카세처럼, 셰프(치수쌤)가 엄선한 웹앱을 코스 요리 형식으로 탐색할 수 있어요.  
각 메뉴에는 시작질문, 웹앱 설명, 키워드, 스크린샷이 담겨 있습니다.

---

## 수록 웹앱

| 카테고리 | 웹앱 |
|---------|------|
| 인문·창작 | 왓츠 인 마이 |
| 수업·학급운영 | 우리반 학급회의 |
| 업무 | 여기 있어 특수문자 |
| 인터랙티브 시각디자인 | 닷 뮤직, 픽셀월드 |
| 디지털·세계시민교육 | 국기 팔레트, SDGs 렌즈 |
| 뮤지컬 | 여기 있어 효과음 |

---

## 기술 스택

- 단일 HTML 파일 (프레임워크 없음)
- [Pretendard](https://github.com/orioncactus/pretendard) · [Noto Serif KR](https://fonts.google.com/noto/specimen/Noto+Serif+KR) 폰트
- [Material Icons Outlined](https://fonts.google.com/icons)

별도의 빌드 도구나 패키지 설치 없이 `index.html` 파일 하나로 실행됩니다.

---

## 로컬 실행
